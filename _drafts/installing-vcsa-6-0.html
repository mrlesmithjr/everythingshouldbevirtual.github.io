Installing VCSA 6.0

In this post we will be going through the initial setup of a new VMware vCenter Server Appliance for 6.0 from the downloaded ISO. You heard it right, the latest version of vCenter is no longer downloaded as an OVF but rather an ISO that you must use to deploy the appliance.
