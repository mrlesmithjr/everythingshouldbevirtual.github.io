Ansible - Building Gerrit Code Review Server

In this post we will be setting up a Gerrit Code Review server to use as our internal Git server which also gives us the ability to do code review. What is the importance of this? This will allow us to house our own Git repositories for code, scripts and etc. Gerrit also gives us the ability to develop code review methodologies in order to automate all of our infrastructure that can be but at the same time give us the ability to have a sign-off on our code.
